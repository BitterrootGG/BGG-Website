# Videos Directory

This directory contains hazard assessment videos for the BitterrootGG website.

## Adding Your Videos

1. Place your hazard assessment videos in this directory
2. Name your files clearly, for example:
   - `hazard-assessment-1.mp4`
   - `hazard-assessment-2.mp4`
   - `site-walkthrough.mp4`

3. The website is already configured to look for videos in this directory

## Supported Video Formats
- MP4 (.mp4) - Recommended
- WebM (.webm)
- OGG (.ogg)

## Recommended Video Specifications
- Resolution: 720p (1280x720) or 1080p (1920x1080)
- Format: H.264 codec for MP4
- Frame rate: 30fps
- File size: Keep under 50MB for faster loading

## Current Video References in index.html

The website is configured to look for these exact filenames:
- `videos/hazard-assessment-1.mp4`
- `videos/hazard-assessment-2.mp4`

**Option 1:** Name your video files exactly as shown above (easiest)
**Option 2:** Update the video paths in `index.html` around line 100 to match your filename
